# H_index
Will be made into a windows Executable and Mac executable

a fun User interface for calculating the H Index with a few additional ancillary features including Altmetrics

If you do not know what the author level bibliometric measure Hirsch Index check out the wikipage : https://en.wikipedia.org/wiki/H-index

FEATURES:

choose your color! auxillary tabs will be colore by its gradient or color complement
